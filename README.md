本工程是一个基于ngbp(v0.3.2)搭建的AngularJS的应用。

ngbp_README.MD介绍了关于ngbp的使用和用到的相应的工具，可以参考该文档或者访问https://github.com/ngbp/ngbp。

本应用主要包含了使用AngularJS的若干示例，可以访问主页面<YOUR_HOME>/index.html来访问。
